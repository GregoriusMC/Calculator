Calculator
==========

This repository stores a very lightweight and simple code that can be used to count numbers.

==========

**Counting Methods**

- Adding 
- Subtracting
- Multiplying
- Dividing

==========

**Counting Symbols**

- **"+"** (Adding)
- **"-"** (Subtracting)
- **"*"** (Multiplying)
- **"/"** (Dividing)

==========
